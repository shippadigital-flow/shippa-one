Roadmap Overview

Shippa One will evolve iteratively, prioritizing customer value, platform stability and simplicity. Releases will favor a usable MVP followed by progressive feature rollouts (Pro features, analytics, integrations, AI recommendations).

Version milestones

- MVP (Current priority)
  - Dashboard
  - Website
  - Blog
  - Settings
  - Library
  - Focus: stability, onboarding and basic publishing

- Version 1.1
  - Customer experience improvements: onboarding, guided tours, empty states, notifications

- Version 2.0 (Analytics — Pro)
  - Traffic, pages, conversions, device & geographic insights, aggregated reports

- Version 2.1 (SEO — Pro)
  - SEO health score, metadata analysis, keyword suggestions, technical recommendations

- Version 2.2 (Leads — Pro)
  - Lead capture, contact management, source tracking, pipeline overview

- Version 3.0+ (Integrations & Smart Reports)
  - Google integrations, Smart Reports, scheduled exports, AI-assisted recommendations

Prioritization principles

1. Customer value
2. Product simplicity
3. Technical feasibility
4. Security
5. Scalability
6. Performance

Success metrics

- Active users, retention, upgrade rate, articles published, feature adoption, CSAT
